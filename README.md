# The Traveling Teddy Co. — Stuffed With Love

Website for **The Traveling Teddy Co.** — mobile bear-stuffing parties that bring the magic to you.

## Tech Stack

- [Next.js 16](https://nextjs.org) (App Router)
- [React 19](https://react.dev)
- [Motion](https://motion.dev) (Framer Motion)
- [Three.js](https://threejs.org) (PixelSnow effect)
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
npm start
```

## Deploy

Deploy on [Vercel](https://vercel.com) or your preferred platform.
